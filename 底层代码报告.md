## 二、初始化：`__init__()`

```python
def __init__(self, eps=None, q_scale=5.0, kernel='invquad',
             anisotropic=True, regression_method='local_linear'):
```

**五个参数，全部可选，零超参数即可运行。**

| 参数 | 默认 | 含义 |
|------|------|------|
| `eps` | None | 背景涨落。None=自动涌现 |
| `q_scale` | 5.0 | 因果荷强度缩放 |
| `kernel` | 'invquad' | 场域函数（决定因果宇宙的物理法则） |
| `anisotropic` | True | 是否启用各向异性维度权重 |
| `regression_method` | 'local_linear' | 回归模式 |

**设计原则**：所有属性（`X_`, `y_`, `q_`, `lambda_`, `r_`, `dim_weights_`）初始化为 `None`。第一次 `place()` 时才分配。这是"延迟构造"——空间在样本落入那一刻才诞生。

**场域函数字典**：

```python
'invquad': lambda d, lam: 1.0 / (1.0 + (d / lam) ** 2),   # 逆二次衰减（默认）
'cos':     lambda d, lam: np.cos(d / lam),                 # 全息震荡
'linear':  lambda d, lam: np.maximum(0, 1.0 - d / lam),    # 紧支撑（纯记忆）
'exp':     lambda d, lam: np.exp(-d / lam),                # 指数衰减
```

同一个框架，换一个核函数，换一个因果宇宙。

## 三、场构造：`place()` → `_recompute_all_props()`

### 3.1 `place(X, y)` — 入口

```python
def place(self, X, y):
    # 1. 标准化
    X_norm = (X - self.x_mean_) / self.x_std_

    # 2. 存储坐标和标签
    self.X_ = X_norm
    self.y_ = y

    # 3. 判定是分类还是回归
    self.is_classifier_ = 自动检测

    # 4. 重算全场属性 ← 空间在这一步被弯曲
    self._recompute_all_props()
```

**设计要点**：
- 第一次 `place` 时记录 `x_mean_` 和 `x_std_`，后续 `place_one` 增量时用同样的归一化
- 自动检测分类/回归：浮点标签 + 唯一值 < 20 → 回归，否则 → 分类
- 全量距离矩阵 `_cdist(self.X_, self.X_)` 是 `O(n²d)`，只在 `place` 时算一次

### 3.2 `_recompute_all_props()` — 核心

这是整个框架的心脏。**空间在这一步被弯曲。**

```
步骤 1：计算各向异性维度权重 w_k（如果启用）
        _compute_dim_weights()  →  self.dim_weights_

步骤 2：计算全距离矩阵
        dists = _cdist(self.X_, self.X_)    ← O(n²d)

步骤 3：从距离推 λ_i（特征长度）
        knn = 每样本到最近 10 个邻居的平均距离
        低维：λ = knn / 全场中位距离（相对密度归一化）
        高维：λ = knn（绝对尺度）

步骤 4：从邻域推 q_i（因果荷）
        在 λ_i 范围内：
          同类占比 → q_i = 1.0 + q_scale × 同类比例 × log(1+邻居数)

步骤 5：计算 ε（背景涨落，如果未手动设置）
        ε = median(q) / 10 ÷ max(1, d/5)

步骤 6：计算 r_i（场域半径）
        r_i = λ_i × √(q_i / ε − 1)
```

**为什么空间在这一步被弯曲？** 因为 `q_i`、`λ_i`、`r_i` 三个数组就位后，空间中任意点的场值已经由场方程隐含确定了：

```
Φ(x) = Σ_i q_i / (1 + ||x−x_i||² / λ_i²)
```

不需要显式计算场——q_i 和 λ_i 就是场的参数化表示。

### 3.3 `_compute_dim_weights()` — 各向异性

```python
对每个维度 k：
  F_k = 类间方差 / 类内方差           ← F 统计量
  w_k ∝ F_k ^ power                   ← 幂律放大
  power = 1.0（d≤100）→ 2.2（d≥500）

归一化：mean(w) = 1.0
裁剪：w ∈ [0.001, 1000]
```

高维时 power 自动增大，弱信号维度被压制到 ~0，强信号维度放大到 10–100×。

### 3.4 `_cdist(A, B)` — 加权距离

```python
if anisotropic and dim_weights_ is not None:
    sqrt_w = sqrt(dim_weights_)
    return cdist(A * sqrt_w, B * sqrt_w)    ← 加权距离
return cdist(A, B)                          ← 普通距离
```

通过预乘 `sqrt(w)` 巧妙实现加权欧氏距离：
`Σ w_k·(a_k−b_k)² = ||a·√w − b·√w||²`

### 3.5 `place_one(x, y)` — 增量学习

```python
def place_one(self, x, y_val):
    # 1. 新样本加入
    self.X_ = vstack([self.X_, x_norm])
    self.y_ = append([self.y_, y_val])

    # 2. 计算新样本的 q, λ, r
    q, lam, r = _local_props_from(x_norm, y_val)

    # 3. 扰动邻域 ← 新样本弯曲了周围空间
    affected = 距离 <= 旧r + 新r 的所有样本
    for each affected:
        重算它的 q, λ, r
```

**每次 `place_one` 不是全量重建——只扰动新样本的邻域。** 这实现了真正的在线学习。

## 四、场读取：`read()`

### 4.1 分类读取：`_read_classify()`

```python
def _read_classify(self, X_norm):
    # 1. 距离矩阵：查询点 × 所有训练样本
    dists = _cdist(X_norm, self.X_)           ← O(mnd)

    # 2. 场核权重矩阵 W
    for each 训练样本 j:
        if 查询点 i 在 j 的场域半径内:
            W[i,j] = q_j × kernel(dists[i,j], λ_j)

    # 3. 跨类干涉
    for each 类别 k:
        score_k = Σ_{同类} W[i,j] − Σ_{异类} W[i,j]/(K−1)

    # 4. 选出得分最高的类
    return argmax(scores)
```

**核心机制**：同类贡献正场（+"引力"），异类贡献负场（−"斥力"）。正负场叠加干涉，得分最高的类胜出。score=0 的地方就是类别边界。

### 4.2 回归读取：`_predict_local_linear()`

```python
def _predict_local_linear(self, X_norm, dists):
    for each 查询点 i:
        # 1. 找到场域覆盖 i 的所有训练样本
        cov_idx = 距离 <= r_j 的样本

        # 2. 如果样本不够，退化到加权平均
        if k < d+2:
            return 加权平均

        # 3. 局部加权最小二乘
        中心化：X_centered = X_local − x_i
        设计矩阵：D = [√w, √w ⊙ X_centered]   ← 截距列 + d 维偏移
        正规方程：(D^T D + ridge·I) β = D^T (y·√w)
        预测：ŷ = β₀                           ← 截距 = 在 x_i 处的值
```

**为什么 β₀ 就是预测值？** 设计矩阵 D 的第一列是截距列 √w，后面是偏移项。在 x_i 处（原点），偏移全为零，只有截距贡献。

## 五、概率输出：`predict_proba()`

```python
def predict_proba(self, X):
    # 1. 和分类读取完全相同的 score 计算
    scores[k] = Σ_{同类} W − Σ_{异类} W/(K−1)

    # 2. softmax 归一化
    scores -= max(scores)                    ← 数值稳定
    exp_scores = exp(scores)
    return exp_scores / sum(exp_scores)      ← 概率 ∈ [0,1]
```

**和逻辑回归的 `predict_proba` 完全一致的接口。** 返回 `(n_samples, n_classes)` 概率矩阵。

## 六、数据流完整追踪

以一个 Iris 分类为例：

```python
sn = SpatialNetwork()                    # __init__: 参数就位
sn.place(X_train, y_train)               # place → _recompute_all_props
    │
    ├─ X_norm = (X-mean)/std             # 标准化
    ├─ _cdist(X_norm, X_norm)            # 105×105 距离矩阵
    ├─ dim_weights_ = [0.22, 0.08,       # w_k: 花瓣是因果主维度
    │                   1.88, 1.82]
    ├─ lambda_ ≈ 每样本邻域尺度          # λ ∈ [0.3, 6.0]
    ├─ q_ ≈ 局部同类密度                # q ∈ [7.5, 18.8]
    ├─ r_ = λ·√(q/ε−1)                  # r ∈ [1.2, 6.7]
    └─ 完成。空间已弯曲。

y_pred = sn.read(X_test)                 # read → _read_classify
    │
    ├─ _cdist(X_test, X_train)           # 45×105 距离矩阵
    ├─ W[i,j] = q_j × 1/(1+d²/λ²)       # 场核权重
    ├─ score_k = Σ同类 W − Σ异类 W/2     # 跨类干涉
    └─ argmax(score) → 预测类别

proba = sn.predict_proba(X_test)         # predict_proba
    │
    ├─ 和 read 相同的 score 计算
    └─ softmax(score) → 概率值
```

## 七、设计原则总结

| 原则 | 实现 |
|------|------|
| **零超参数** | `eps=None`, `q_scale=5.0` 开箱即用 |
| **自动检测分类/回归** | `is_classifier_` 从标签类型自动判定 |
| **一次性完成** | `place()` 一次调用，空间弯曲完成 |
| **增量学习** | `place_one()` 只扰动邻域，不重算全场 |
| **延迟构造** | 属性初始为 None，第一次 place 时分配 |
| **场核可插拔** | `kernel='invquad'/'cos'/'linear'/'exp'` 或自定义函数 |
| **各向异性默认启用** | 高维时自动放大维度权重差异 |
| **概率接口兼容** | `predict_proba` 和 sklearn 完全一致 |
