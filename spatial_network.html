三、概率输出 —— ROC / 分布 / 散点

Breast Cancer 二分类。SN predict_proba vs 逻辑回归 predict_proba。

左：ROC 曲线——SN AUC=0.989，LR AUC=0.998。差距 0.009。

中：SN 概率倾向于极端值（0 或 1）——场干涉是竞争性的，赢家通吃。LR 概率分布更平滑。

右：每个点是一个测试样本。x=LR概率，y=SN概率。对角线附近说明两者判断一致。

四、决策边界 —— 线性可分 / 双月形

2D 数据上 SN / LR / SVM 的决策面可视化。红色/蓝色 = 两个类别。

SN 的边界来自场干涉——正负因果荷抵消处的因果静区。不同于 LR 的线性超平面和 SVM 的核化边界。

五、训练效率 —— 随样本数增长

30D 数据，对比 SN / LR / SVM / RF / MLP 的训练时间。

SN 无迭代——place() 是一次 O(n²d) 距离矩阵计算。小数据时最快，大数据时 O(n²) 增长。传统 ML 的迭代成本随数据量和复杂度增长。
